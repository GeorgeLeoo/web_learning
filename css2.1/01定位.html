确定包含块

    确定一个元素的包含块的过程完全依赖于这个元素的 position 属性：
    1.  如果 position 属性为 static 或 relative ，包含块就是由它的最近的父级块元素（比如
        说inline-block, block 或 list-item元素）或 格式化上下文(比如说 a table container,
        flex container, grid container, or the block container itself)的
        内容区的边缘组成的。

        内边距的边缘组成。
